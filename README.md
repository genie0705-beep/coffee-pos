# ☕ 커피숍 POS - 설치 및 배포 가이드

## 📁 파일 구성

```
coffee-pos/
├── index.html      ← 메인 앱 (POS 시스템)
├── manifest.json   ← PWA 설정 파일
├── sw.js           ← 오프라인 지원 (서비스워커)
├── icons/
│   ├── icon-192.png
│   └── icon-512.png
└── README.md       ← 이 파일
```

---

## 🚀 배포 방법 (무료 - GitHub Pages)

### 1단계: GitHub 가입
- https://github.com 접속 → Sign up (무료)

### 2단계: 저장소 만들기
1. 로그인 후 우측 상단 **+** → **New repository**
2. Repository name: `coffee-pos` 입력
3. **Public** 선택
4. **Create repository** 클릭

### 3단계: 파일 업로드
1. 생성된 저장소 페이지에서 **uploading an existing file** 클릭
2. 압축 해제한 모든 파일/폴더를 드래그 앤 드롭
   - index.html, manifest.json, sw.js, icons 폴더
3. **Commit changes** 클릭

### 4단계: 웹사이트 활성화
1. 저장소 → **Settings** 탭
2. 좌측 메뉴 **Pages** 클릭
3. Source: **Deploy from a branch** 선택
4. Branch: **main** / **/ (root)** 선택 → **Save**
5. 1~2분 후 URL 생성됨:
   **https://내아이디.github.io/coffee-pos/**

## 📱 폰/태블릿에 앱으로 설치

### iPhone / iPad (Safari)
1. Safari에서 배포된 URL 접속
2. 하단 **공유 버튼** (↑) 탭
3. **홈 화면에 추가** 선택
4. **추가** 탭 → 완료!

### Android (Chrome)
1. Chrome에서 배포된 URL 접속
2. 상단에 **"앱 설치"** 배너가 뜨면 탭
   (안 뜨면: 메뉴 ⋮ → **앱 설치** 또는 **홈 화면에 추가**)
3. **설치** 탭 → 완료!

설치 후 홈화면에 커피 아이콘이 생기며, 전체화면 앱처럼 실행됩니다.

## 💡 사용 안내

### 주문 프로세스
1. 메뉴 카테고리 선택 (커피/논커피/차/음료)
2. 음료 탭하여 장바구니 추가
3. 텀블러 고객이면 🥤 버튼으로 해당 음료에 할인 적용
4. **결제하기** → 현금/계좌이체 선택
5. 현금: 받은 금액 입력 → 거스름돈 자동 계산
6. **진동벨 번호 선택** → 주문 완료

### 관리자 기능
- 우측 상단 **📊 관리자** 버튼
- 날짜별 매출 조회 (◀ ▶ 이동)
- 주문내역 / 시간대별 / 메뉴별 탭

### 데이터 저장
- 모든 주문 데이터는 브라우저 로컬 저장소에 보관됩니다
- 브라우저 데이터를 삭제하면 주문 기록도 삭제되니 주의하세요
- 같은 기기, 같은 브라우저에서만 데이터가 유지됩니다
